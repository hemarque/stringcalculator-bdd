Numbers bigger than 1000 should be ignored.

So adding 2 + 1001 = 2.

Examples

Given an Add method on String Calculator
When numbers contains a number bigger than 1000 like "1\n1002"
Then the execution result is 1

Given an Add method on String Calculator
When numbers contains a number bigger than 1000 like "3\n1000\n1006"
Then the execution result is 1003
