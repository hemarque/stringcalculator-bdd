Allow multiple delimiters

Allow multiple delimiters like this: “//[delim1][delim2]\n” for example “//[*][%]\n1*2%3” should return 6.

Examples

Given an Add method on String Calculator
When numbers contains new delimiters like "//[;][a]\n1;a2"
Then the method returns 3

Given an Add method on String Calculator
When numbers contains new delimiters like "//[a][b][c]\n1a2b3c1"
Then the method returns 7

Given an Add method on String Calculator
When numbers contains a new delimiter like "//[*][;]\n1*2;3"
Then the method returns 6
