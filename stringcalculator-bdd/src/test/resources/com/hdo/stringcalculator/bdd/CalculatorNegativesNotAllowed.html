Calling Add with a negative number will throw an exception “negatives not allowed”

Calling Add with a negative number will throw an exception “negatives not allowed” - and the negative that was passed.if there are multiple negatives, show all of them in the exception message.

Examples

Given an Add method on String Calculator
When numbers contains a new delimiter like "//;\n1;-2"
Then the method throws an Exception -2

Given an Add method on String Calculator
When numbers contains a new delimiter like "//a\n-1a2a-3"
Then the method throws an Exception with the numbers-1 -3
